MINHAS ESPECIALIDADES.

HTML5

HTML5 é a tecnologia que dá estrutura e significado ao conteúdo do seu portfólio. Ele permite organizar textos, imagens, vídeos e seções de forma semântica e acessível, além de garantir compatibilidade com navegadores modernos e facilitar a integração com CSS e JavaScript para criar páginas profissionais e interativas.

CSS3

CSS é a linguagem usada para estilizar e dar identidade visual ao seu portfólio. Com ele você define cores, fontes, layouts responsivos e animações, transformando a estrutura criada em HTML em uma interface atraente, moderna e profissional.

Java Script

JavaScript é a linguagem que traz interatividade e dinamismo ao seu portfólio. Ele permite criar animações, validações, efeitos visuais e funcionalidades inteligentes, tornando a navegação mais envolvente e moderna para o usuário.
